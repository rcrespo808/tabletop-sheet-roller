# Handout Reward Test Checklist

## Claiming the Rat Chapel Bounty

1. Open `/handouts`.
2. Import starter handouts.
3. Select `Wanted Poster: The Rat Chapel Heretic`.
4. Confirm it is visible to the campaign.
5. Select `He Zhen` as the target character.
6. Open the reward preview.
7. Verify the preview shows:
   - `+75 Gp`
   - `Condition: Wanted by Veinwatch`
   - `Note: Veinwatch Recognition`
8. Confirm apply.
9. Open He Zhen's character sheet and verify:
   - wallet `gp` increased by 75
   - conditions include `Wanted by Veinwatch`
   - reward history includes source `Handout: Wanted Poster: The Rat Chapel Heretic`
10. Return to `/handouts` and verify the handout detail shows `Applied to He Zhen`.
11. Check the roll log for campaign/system entries for the reveal or reward application.
